## Tech Stack

**Client:** HTML, JS, Bootstrap5

**Server:** Python, Django

## Run Locally

Clone the project

```bash
  git clone git@github.com:zablon-oigo/listings.git
```

Go to the project directory

```bash
  cd listings
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  py manage.py runserver
```
